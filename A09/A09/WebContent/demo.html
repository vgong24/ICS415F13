Flickr Sidebar

Just a post Title

May 8, 2011 in Category

Copy the .html files to the A08 directory; the .css files to the A08/css directory. Edit demo-step1.html and final.html to point to the correct css directory for the style sheets. As you go through the tutorial pay particular attention to the styles applied to each element. Use the “Inspect Element” feature to see the styles applied to the elements. Change the size of your browser and see the affects on the styles. Think about these questions while going through the CSS files: In ‘wide’ screen mode, what causes the content and sidebar to be side-by-side? In the ‘fluid’ mode, what are the widths of the content and sidebar How does the CSS Media Query force a single column layout for narrow screens? How would you support more than two columns in the ‘wide’ and ‘fluid’ mode? Try and add a left sidebar and move the Flickr widget into it. What is the correct behavior for the Flickr widget in the one column mode? (Should it be above the contents or below?) The Responsinator is a valuable tool for viewing your web site in different widths. Important note for Chrome users: you must start Chrome with the -allow-file-access-from-files command line switch in order to load style files and javascript associated with local files.
